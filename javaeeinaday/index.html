Book Description

Using a new interactive approach this book is teaching you the essential components of the Java Enterprise Edition.
You start with the most simple application consisting of just a static HTML file. Using WildFly, a certified Java EE 7 application server, you learn how to serve this file.

Step by step we enhance this simple project to cover the core APIs: Servlets, JSP, JSF, CDI and EJB.
Knowing the Java Standard Edition you can go through this tutorial in just a day. To ensure that you’re not bothered with any configuration you will use Codenvy, an Online IDE.

Approx. 30 pages (given 440 words per page).
