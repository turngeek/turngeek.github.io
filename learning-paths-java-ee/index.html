At TurnGeek you can find professional IT books that are easy to understand and fully accessible online. In our Cloud Tutorials, we make use of modern technologies like Cloud IDEs, so you can immediately jump start without any previous configuration.

Currently, our offer is about Java EE. Interested? Our learning paths explain how to start best in your personal situation.

Learning Paths for Java EE

 

  1. Quickstart! – You want to get a quick overview of the Java EE programming platform and its major technology components? Recommendation: Start with Cloud Tutorial Java EE in a Day.
  2. More! – You want to deep dive into the Java EE programming platform (e.g. after you had the quickstart)? Recommendation: Start with Cloud Tutorial Series Java EE and work of the series sequentially.
  3. Focus! – You have already experiences with Java EE  but want to focus or repeat on a certain Java EE technology? Recommendation: Start directly with the corresponding Cloud Tutorial. E.g. if your interessted in JSF, start with Cloud Tutorial JavaServer Faces in a Day.
