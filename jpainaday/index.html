Book Description

The Java Persistence API is the Java standard for object-relational mapping. Version 2.1 of the Java Persistence API (JPA) was included in Java EE 7 for persisting entities in a database. JPA enables us to store instances of entity classes in a relational database and re-trieve them from the database at a later date.
